# Chat
Chat app. Working in real-time by websockets.

This project was reviewed by Hexlet.

[Link to deployed project](https://secure-everglades-75385.herokuapp.com/)

### Features:
- Creating messages
- Creating channels
- Deleting channels
- Renaming channels

### Used in project:
- **react** + **hooks**
- **redux**
- **socket.io**
- **bootstrap**
- **formik**
- **axios**
- **faker**
- **js-cookie**
- **webpack**
- deploy on **heroku**
